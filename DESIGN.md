# Dipansu Web Design Brief

## Purpose & Tone
Professional freelance services landing page for video editing and thumbnail design. Premium, modern, energetic without being frivolous. Target: creative professionals and content creators seeking high-quality design services.

## Palette (Dark Mode)
| Token | OKLCH | Usage |
|-------|-------|-------|
| Background | 0.12 0 0 | Page background (navy #0f172a) |
| Card | 0.18 0.02 250 | Service cards, sections (slate #1e293b) |
| Foreground | 0.95 0.01 250 | Primary text (off-white) |
| Muted | 0.62 0.04 250 | Secondary text (slate) |
| Primary/Accent | 0.68 0.2 142 | CTA buttons, highlights (emerald #22c55e) |
| Border | 0.38 0.03 250 | Dividers, subtle edges |

## Typography
| Layer | Font | Usage |
|-------|------|-------|
| Display | General Sans (600) | Headlines, section titles |
| Body | DM Sans (400) | Body copy, descriptions, UI text |
| Mono | Geist Mono (400) | Pricing, technical details |

## Structural Zones
| Zone | Treatment | Purpose |
|------|-----------|---------|
| Header | Card bg (0.18), sticky, shadow-subtle | Navigation, logo, fixed CTA |
| Hero | Background + gradient accent text | Headline, tagline, large CTA |
| Services | Card grid (0.18 bg), shadow-elevated on hover | Service offerings with price/description |
| Differentiators | Background with icon accents | Trust/quality indicators |
| CTA Section | Muted bg (0.22), centered layout | Contact call-to-action |
| Footer | Background with border-t | Copyright, links |

## Shape Language
- Border radius: 0.5rem (sm: 0.375rem, md: 0.25rem)
- Shadows: subtle (2px, 0.2 alpha) for cards, elevated (4px, 0.4 alpha) on hover
- Spacing: 2rem (header/footer), 3rem (sections), 1.5rem (cards)

## Component Patterns
- Buttons: Primary (bg-primary, text-primary-foreground, rounded-md), hover scale + shadow
- Cards: bg-card, shadow-subtle, border-border, rounded-lg
- Links: text-primary, underline on hover, transition-smooth

## Motion & Micro-interactions
- Transition: smooth (0.3s cubic-bezier)
- Button hover: scale(1.02) + shadow-elevated
- Card hover: translate-y(-2px) + shadow-elevated
- Section fade-in on scroll (optional, enhanced perception)

## Constraints
- NO rounded-full for non-iconography
- NO gradient backgrounds on large surfaces (use solid + text gradients sparingly)
- NO animations without purpose
- All CTAs green (#22c55e) linking to WhatsApp
- Maintain AA+ contrast across all text

## Signature Detail
Green accent used surgically — primary buttons, section highlights, hover states — creating cohesive energy without visual noise. Professional restraint paired with premium tech aesthetic.
